# 리플래시 근태관리 (Reflash T&A)

주식회사 리플래시 사내 근태관리 웹앱입니다.

- 직원 출퇴근 기록 (키오스크 방식)
- 일별/주별/월별 근태 현황, 지각·조퇴·결근 자동 판정
- 급여 자동 계산 (월급제 209시간 · 3.3% 사업소득 · 일용직 시급제)
- 급여 관리대장 엑셀 다운로드 (관리자)

## 사용 방법
1. 접속 후 [직원관리] 탭(관리자 비밀번호 필요)에서 직원을 등록하거나,
   [설정] 탭 → [백업 복원]으로 기존 데이터(JSON)를 불러옵니다.
2. 데이터는 사용하는 기기(브라우저)에만 저장됩니다.
   기기를 옮길 때는 [설정] → [백업 다운로드] → 새 기기에서 [백업 복원]을 이용하세요.

※ 이 저장소 버전에는 개인정보(직원 정보)가 포함되어 있지 않습니다.
